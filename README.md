# Dependancies

1. NodeJS
2. XCode and XCode command line
3. Macports
* ImageMagick
* curl-ca-bundle

# Install

npm install

# Setup

Download adapt2 zip file to root dir

# Run

node adapt-to-app <zipfile.zip>

# Sign and run on device

node adapt-to-app.js dlab-video-in.zip run --keystore=odi2.keystore --keyalias=ODI
